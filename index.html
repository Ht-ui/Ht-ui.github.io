项目概述

系统要求

确保您的设备满足以下要求以获得最佳使用体验

硬件与软件要求

操作系统

Windows 10/11, macOS 10.14+, Linux (Ubuntu 20.04+)

Python环境

Python 3.6 或更高版本，需确保Python已添加到系统PATH

内存要求

最少 2GB RAM，建议4GB以上以获得更流畅的体验

存储空间

取决于视频库大小，建议预留足够空间存放视频文件

网络环境

局域网环境，确保设备在同一网络下

防火墙配置

需要开放8080端口，允许局域网设备访问视频服务器

重要提示

在开始部署前，请确保已正确安装Python并配置了系统环境变量。如果使用Windows系统，还需要配置防火墙允许8080端口的入站连接。

环境部署详细要求

Python安装与验证

访问Python官方网站下载最新版本，安装时务必勾选"Add Python to PATH"选项。

# 验证Python安装
# 打开命令提示符，输入以下命令：
python --version
# 如显示Python版本（如Python 3.9.0），则安装成功

防火墙端口配置

按Win+R键，输入wf.msc打开防火墙高级安全设置，创建8080端口的入站规则。

端口测试

部署完成后，可以使用浏览器访问http://localhost:8080测试服务器是否正常运行，其他设备可以访问http://[服务器IP地址]:8080进行测试。

三步部署指南

按照以下步骤快速完成视频库服务器的部署

步骤1：环境部署

在开始项目部署前，需要确保系统环境已准备就绪，包括Python安装和防火墙配置。

重要备注

您也可以选择直接关闭系统防火墙，但这会降低系统的安全性，强烈不推荐这样做。正确配置防火墙端口规则是更安全、更专业的方法。

视频教程参考

  • Python安装教程
  • Windows端口测试和防火墙管控

步骤2：项目部署

创建项目目录

在您选择的目录下创建项目文件夹，目录名称可以为任意英文字母，例如：VideoLibrary。

# 示例：在D盘创建项目目录
mkdir D:\VideoLibrary
cd D:\VideoLibrary
重要提示

为了兼容性考虑，不推荐使用中文、空格或特殊字符作为目录名。

下载项目文件

下载以下核心项目文件到创建的目录中：

  • video_server.py - 主服务器文件，处理所有后端逻辑和API接口
  • start_server.bat - Windows启动脚本，简化服务器启动流程
  • video_library.html - 前端用户界面，提供视频播放和浏览功能
  • 使用手册.html - 本使用手册的网页版，方便用户随时查阅

步骤3：视频添加

初始化项目结构

运行启动脚本：双击 start_server.bat，系统将自动生成以下项目结构：

项目目录/
├── video_server.py # 主服务器文件
├── start_server.bat # Windows启动脚本
├── video_library.html # 前端用户界面
├── 使用手册.html # 本使用手册的网页版
├── videos/ # 视频文件存储目录（自动生成）
├── thumbnails/ # 缩略图缓存目录（自动生成）
└── video_library.db # SQLite数据库文件（自动生成）

添加视频文件

将MP4格式的视频文件放入 videos 文件夹，启动服务器后，在浏览器中访问视频库，然后点击批量添加功能添加视频。

注意

当前版本不支持二级目录，请将所有视频文件直接放在videos目录下。

使用教程

掌握视频库的各项功能，高效管理您的视频资源

启动与访问

启动服务器

运行启动脚本：Windows用户双击 start_server.bat，其他系统在命令行中执行 python video_server.py。

访问视频库

  • 本机访问： 在浏览器中输入：http://localhost:8080
  • 局域网访问： 其他设备访问：http://[服务器IP地址]:8080

核心功能

批量添加视频

将视频文件移动到项目视频目录 videos 内，在视频库页面点击批量按钮，填写对应的年份和番组，选择需要添加的视频，点击保存按钮执行批量添加。

注意

只能添加项目视频目录内的视频，且不支持二级目录。

编辑模式

点击页面顶部的编辑按钮进入编辑模式，支持全选、单选、批量编辑和批量删除功能。

删除说明

删除功能执行的是软删除，删除视频卡片后项目视频目录内的视频不会被删除，仍然会保留在视频目录中。这是为了避免误删而设计的方案，但如果硬盘容量告急，记得适当清理视频目录的视频资源。

搜索与筛选

添加视频后，系统会根据视频的年份、番组等属性生成标签，点击对应的标签可筛选视频。在搜索框中输入关键词可按视频标题进行搜索，支持模糊搜索。

单独编辑与删除

鼠标悬停在视频卡片上，右上角会显示编辑按钮和删除按钮。编辑功能支持修改视频标题、年份、番组以及自定义缩略图。删除按钮可删除单个视频卡片（软删除）。

故障排除

遇到问题？请参考以下解决方案

常见问题与解决方案

1. 服务器无法正常启动

  • Python未正确安装： 检查Python是否安装，在命令行输入 python --version
  • 端口被占用： 检查8080端口是否被其他程序占用，常用命令：netstat -ano | findstr :8080
  • 防火墙阻止： 确认防火墙已正确配置8080端口规则

2. 视频无法播放

  • 视频格式不支持： 确认视频格式为MP4，可使用格式转换工具转换为H.264编码的MP4
  • 文件路径错误： 确认视频文件在 videos 目录下
  • 浏览器兼容性问题： 尝试使用Chrome、Edge等现代浏览器

3. 局域网设备无法访问

  • 防火墙设置问题： 确认已创建8080端口入站规则
  • IP地址错误： 确认使用的是正确的服务器IP地址
  • 网络连接问题： 确认所有设备在同一局域网

4. 视频播放卡顿，黑屏，无加载

  • 客户端设备兼容性问题： 检查播放设备使用的浏览器是否为最新版本，尝试更新浏览器
  • 服务器宕机问题： 初始化页面后，第一次启动时，主机端拖拽视频进度条等操作可能导致服务器宕机。此时不要关闭页面，重启服务器即可解决。重启方法：关闭服务器脚本，重新运行 start_server.bat

注意事项

使用前请仔细阅读以下注意事项

重要提醒

视频文件管理

当前版本不支持二级目录，请将所有视频文件直接放在videos目录下。

设备访问限制

系统不支持异步访问，同一时间只能有一个设备进行视频播放操作。

视频格式兼容性

建议使用H.264编码的MP4格式，以确保在所有浏览器中正常播放。

文件命名规范

建议使用英文命名视频文件，避免中文字符可能导致的问题。

定期维护

建议定期清理不常用的视频文件，保持系统运行效率。

删除注意事项

删除视频卡片是软删除，不会删除物理视频文件，如需彻底删除，需手动清理videos目录。

缩略图管理

自定义缩略图会上传到thumbnails目录，如需清理，可删除此目录下的文件。

数据备份

定期备份video_library.db数据库文件，防止数据丢失。
